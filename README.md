# Personal Website

**Live:** [undefined-behaviour.vercel.app](https://undefined-behaviour.vercel.app)

A portfolio site styled as a vinyl record liner notes package — built with vanilla HTML, CSS, and JavaScript, no frameworks.

## Features

- Scroll-synchronized animations with continuous JS interpolation (blur, opacity, progress fader)
- CSS stacking-context-aware layering for card/CD reveal effects
- Scroll-driven mixing board fader as a progress indicator
- GPU-accelerated transitions via `will-change` on animated elements
- Fully responsive single-page layout

## Stack

Plain HTML · CSS · Vanilla JavaScript · Deployed on Vercel
